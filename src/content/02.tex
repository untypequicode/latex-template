\section{Utilisation avancée}

\subsection{Gestion des références bibliographiques}

Le fichier \texttt{references.bib} contient toutes les références utilisées dans ce document. Vous pouvez y ajouter vos propres sources en suivant les exemples fournis.

\subsection{Types de références supportés}

Ce template supporte plusieurs types de références :

\begin{description}
	\item[Articles de journaux] comme \cite{example_article} pour les publications scientifiques
	\item[Livres] tel que \cite{example_book} pour les ouvrages de référence
	\item[Chapitres de livres] voir \cite{example_chapter} pour les contributions spécifiques
	\item[Conférences] comme \cite{example_conference} pour les actes de conférence
	\item[Thèses] telle que \cite{example_thesis} pour les travaux de recherche
	\item[Sites web] comme \cite{example_website} pour les ressources en ligne
\end{description}

\subsection{Personnalisation}

Le style de citation (numérique, auteur-année, etc.) peut être modifié dans le fichier \texttt{preamble.tex} en changeant les options du package \texttt{biblatex}.
